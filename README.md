# Cosmic Time Machine

An interactive web experience that lets you travel through the entire history of the universe, from the Big Bang to heat death.

## Demo

https://khanaaaaaa.github.io/Cosmic-Time-Machine/

## Features

### Interactive Timeline
Scroll through 13.8 billion years of cosmic history across 21 major events including:
- The Big Bang and cosmic inflation
- Formation of first stars and galaxies
- Birth of our solar system
- Present day
- Future events including the sun's death and heat death of the universe

### Dynamic Universe Simulation
- Procedurally generated stars and galaxies that appear and disappear based on timeline
- Smooth transitions between cosmic eras
- 3D space rendered with Three.js

### Cosmic Event Markers
Click glowing markers on the right side to jump to major events:
- Big Bang
- First Stars
- Milky Way Formation
- Solar System Birth
- Present Day
- Heat Death

### Audio Experience
Ambient sound that gradually fades as you scroll toward the heat death of the universe, representing the silence of entropy.

### NASA Integration
Real-time data from NASA's Astronomy Picture of the Day API displays alongside the timeline.

### Stand on a Planet Mode
Click any planet to switch to surface view and experience:
- Gravity differences
- Unique sky colors
- Star brightness variations
- Data from real exoplanets including Kepler-452b, HD 189733b, and TRAPPIST-1e

## Technologies

- Three.js for 3D rendering
- JavaScript ES6+
- NASA APOD API
- HTML5 Canvas
- CSS3 animations

## Installation

1. Clone the repository
```bash
git clone https://github.com/khanaaaaaa/Cosmic-Time-Machine.git
```

2. Open `index.html` in a modern web browser

No build process or dependencies required.

## Usage

- Scroll to travel through time
- Click event markers to jump to specific cosmic events
- Click planets to stand on their surface
- Click anywhere in planet mode to return to space view

## Browser Support

Works best in modern browsers with WebGL support:
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
